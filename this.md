## المفهوم 
تقوم كلمة
`this `.

تشير الكلمة الأساسية إلى المثيل الحالي للفئة وتستخدم أيضًا كمعدل للـ parameter لطريقة الامتداد.

  

## المثال

يوضح المثال التالي كفية استخدام كلمة `enum `:

</div>

  

```C#

public  class  Student

{

private  string  name;

private  int  id;

public  Employee(string  name, string  id)

{
this.name = name;
this.id = id;
}
}
```
